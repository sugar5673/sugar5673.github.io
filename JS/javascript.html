# HTML5 Audio Player Duration 在IOS兼容问题

问题描述

解决办法

  • 增加一个事件监听就可以解决了
var audio = new Audio();
audio.src=""; //audio链接
audio.addEventListener('loadedmetadata',function(){
    //console.log(audio,duration);
})
